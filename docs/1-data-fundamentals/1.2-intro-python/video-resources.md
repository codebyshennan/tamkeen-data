## üí° Tips for Learning from Videos

### Before Watching
- [ ] Have Python Tutor open in another tab
- [ ] Open Google Colab for live coding
- [ ] Prepare note-taking tool
- [ ] Set playback speed to 1.25x-1.5x

### During Watching
- [ ] Pause frequently to try examples
- [ ] Paste code into Python Tutor
- [ ] Take notes on key concepts
- [ ] Screenshot important diagrams

### After Watching
- [ ] Complete related exercises in course materials
- [ ] Ask AI to quiz you on the topic
- [ ] Build a small project using what you learned
- [ ] Teach the concept to someone else (best way to learn!)
